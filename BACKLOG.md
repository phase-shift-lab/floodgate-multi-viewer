# Backlog

P0の実用性・信頼性・公開を優先し、以下は初版完了条件から分離する。

## P1

- PWA（オフラインシェル、インストール）
- ブラウザ通知（注目対局開始・終局、明示許可制）
- 戦型判定（誤判定表示と根拠の設計を含む）
- 4局同期再生

### 視聴者コメント共有（P1候補）

- 対局IDと手数に紐づけた短文コメントを表示し、4局表示では局ごと、単局表示では詳しく閲覧できるようにする。
- 匿名表示名は自動生成し、本文は80文字以内、URL・HTMLは禁止する。
- 連投制限、通報、ミュート、管理者非表示を備える。
- Worker + Cloudflare D1で構築し、5〜10秒ポーリング、IPアドレスは生保存しない。
- 保存期間は7日、1対局あたり最新50件を上限とする。
- `/comments` 系の固定ルートへ分離し、既存のFloodgate限定プロキシを任意URLプロキシへ拡張しない。
